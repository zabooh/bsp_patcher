# BSP Patcher

## Usage

To apply a patch, you can use the command:

```bash
patch -p1 < your_patch_file.patch
```

Replace `your_patch_file.patch` with the name of your patch file. 

## Examples

If you have a patch file named `example.patch`, you can apply it using:

```bash
patch -p1 < example.patch
```

Make sure to run this command in the root of your repository.

## Additional Information

For more details on how patching works, refer to the official documentation of the `patch` command.
